Display information about current COVID-19 in Mexico

== Description ==

You can display a map with information about COVID-19 in Mexico.

== How to use ==

Paste the shortcode below in a page or post:
[hrmex-covid-map]

Optionally set a custom width and/or height in % or px:
[hrmex-covid-map width="300px" height="300px"]
